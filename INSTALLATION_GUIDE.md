# PO Professional Tools v0.1.3 - Installation Guide

## Quick Start

### Prerequisites
- **VS Code:** v1.96.0 or later
- **File:** `po-professional-tools-0.1.3.vsix`

### Installation Steps

1. **Download the VSIX**
   - Download `po-professional-tools-0.1.3.vsix` from the repository

2. **Open VS Code**
   - Launch VS Code

3. **Install from VSIX**
   - Open the **Extensions** view (Ctrl+Shift+X)
   - Click the **⋯** menu button in the top right of the Extensions sidebar
   - Select **Install from VSIX...**
   - Browse to and select `po-professional-tools-0.1.3.vsix`
   - Wait for installation to complete

4. **Verify Installation**
   - Look for **PO Professional Tools** in your Extensions list
   - Status should show as "Installed"

### First Run

After installation, the extension is ready to use:

**Open Command Palette** (Ctrl+Shift+P) and type to run commands:
- `PO Tools: Open Dashboard` - Access the main project dashboard
- `PO Tools: Open PBI Studio` - Open PBI generation studio
- `PO Tools: Open Bulk Breakdown` - Bulk PBI breakdown tool

### Troubleshooting

| Issue | Solution |
|-------|----------|
| Extension not appearing | Reload VS Code (Ctrl+R or Cmd+R) |
| Commands not found | Ensure extension is enabled and VS Code is reloaded |
| Webview panel fails to load | Check VS Code Output → Extension Host for errors |

### System Requirements

- **VS Code:** 1.96.0+
- **Memory:** Minimum 512 MB available
- **Storage:** ~20 MB for extension files

### Performance Note

First load may take 2-3 seconds as the extension initializes. Subsequent loads are faster.
