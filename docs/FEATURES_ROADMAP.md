# Features & Roadmap

---

## ✅ Completed Features

- **Project Management** — Add, scan, remove local code repositories.
- **Code Scanning** — Detect routes, API endpoints, and SQL objects.
- **PBI Generation** — Create initial drafts with effort, acceptance criteria, and test scenarios.
- **In-App Editing** — Edit all PBI fields without leaving the tool.
- **Dashboard View** — See project KPIs and recent activity.
- **Azure DevOps Settings** — Store org URL, project, paths, PAT (securely in VS Code).
- **ADO Push** — Send backlog items to Azure DevOps with the correct work item type.
- **Copilot Refinement** — Round-trip AI suggestions or hand off to Copilot Chat.
- **Bulk Breakdown** — Create many prefixed child items from manual input, AI suggestions, or scanned code.
- **Theming** — Light, Dark, or Auto theme (persisted per user).
- **Push Tracking** — See which drafts have been pushed and their ADO status.

---

## 🔄 In Progress

- **UX Overhaul** — Improved wizard UI with step indicators, better form layout.
- **Bug Report Wizard** — Specialized form for filing bug reports with INVEST checklist.
- **Collapsible Sections** — Organize utility features (Edit, Refine, Chat) into collapsible cards.

---

## 📋 Planned (Not Started)

- **Test Case Work Items** — Create dedicated ADO Test Case items linked to PBIs.
- **Copilot Chat Participant** — Register as `@po-tools` in VS Code Chat for persistent context.
- **Process Template Mapping** — Support different ADO process templates (Scrum, Agile, CMMI).
- **Kanban Board** — View and update Azure DevOps board state directly in the extension.
- **Iteration Planner** — Plan and estimate across sprints without switching to ADO.
- **Better Parser Coverage** — Improve scanning accuracy for React, .NET, minimal APIs, and SQL.
- **Marketplace Publishing** — Simplify install and updates via Visual Studio Marketplace.
